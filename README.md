# 河岸雪仗 · Snow Type Battle

一个隔着冰河、靠英文打字抢雪花的童年小游戏重制版。角色采用原创 CSS 绘制，只参考 Snowcraft 圆润、厚实的冬装小人气质，不使用原作精灵或素材。

## 当前玩法

- 两队人数可分别设置为 1–4 人，因此支持 1v1、1v2、2v3、4v4 等非对称阵容。
- 每个角色拥有独立血量；每次雪球都锁定敌方最前排的存活角色，前排倒下后才会打到下一位。
- 阵型可以在开战前调整，让高血量肉盾挡在前面、输出角色留在后排。
- 所有单词雪花都是英文，只接受 `a-z` 输入；雪花落地后会一直保留，直到被玩家或 AI 抢走。
- 雪花有三种竞速状态：无人输入、AI 正在输入、玩家当前领先。
- 每个 AI 席位都可独立选择新手、熟练或高手强度。
- 抢到单词后会完整播放抓取、攥雪球、蓄力投掷、飞行和受击动画。

当前可玩版本是本机 AI 战术演练：1 位真人加入雪松队，其余活动席位由可调强度 AI 补齐。它不是伪装成联网的真人房间。

## 真正联机的下一步

推荐让网页继续作为客户端，并新增一个权威 WebSocket 房间服务：每个房间最多 8 个席位，由服务器统一生成单词、裁定谁先打完、选择前排目标、扣血和宣布胜负。客户端只发送输入与准备指令，并按服务器事件播放现有动画。Cloudflare 环境下可用 Worker + Durable Object 实现“一房间一个状态实例”，并加入房间码、重连快照、消息序号和服务端限速。
